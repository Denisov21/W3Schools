# W3Schools


This is an extension created for [Brackets 42 or later](http://www.brackets.io/). 

It adds a link:

* In `Help` menu
[W3School](http://www.w3schools.com/), [MDN](https://developer.mozilla.org/) 
* In `View` menu
[Brackets Themes](http://brackets-themes.github.io/)
